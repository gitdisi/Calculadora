# Calculadora Web Interativa

Uma calculadora web desenvolvida em JavaScript, com interface interativa e funcionalidades básicas de operações aritméticas. O projeto utiliza manipulação do DOM, controle de eventos e boas práticas de programação.

## Funcionalidades

- **Operações Básicas:**  
  - Adição, subtração, multiplicação e divisão.
  - Tratamento de números decimais e porcentagens.

- **Interface Interativa:**  
  - Atualização dinâmica do display por meio de botões.
  - Botões de backspace e limpeza para facilitar a edição.
  - Bloqueio da digitação direta no input, garantindo que a interação seja feita apenas via botões.

- **Recursos Adicionais:**  
  - Efeito sonoro no botão do sapinho para uma experiência mais divertida

## Tecnologias Utilizadas

- **Linguagens:** HTML, CSS, JavaScript  

## Instalação e Execução Local

1. Clone o repositório:
   ```bash
   git clone https://github.com/gitdisi/Calculadora
   ```
   
2. Abra o arquivo `index.html` no seu navegador para utilizar a calculadora.

## Considerações

- **Responsividade:**  
  O layout foi desenvolvido para proporcionar uma boa experiência tanto em desktops quanto em dispositivos móveis.

- **Boas Práticas:**  
  O projeto utiliza arrays para gerenciamento dos tokens de entrada, funções modulares para processamento de operações e tratamento de exceções para garantir robustez na aplicação.
